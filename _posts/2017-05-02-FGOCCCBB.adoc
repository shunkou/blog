= 【FGO】CCCコラボ(BBちゃんの逆襲)メモ

== 1. クエスト解放
- キーを交換することで解放
- サクラマネーで交換
- ミッション報酬で入手

=== 効率良いミッション
cf. https://game8.jp/fate-go/146250

- No.56 「頭クエストを合計10回クリア」
- No.8 「TYPEⅠの敵を30体撃破」
- No.37 「「秩序の敵を10体撃破」」

== 2. 概念礼装「始まりの予感」回収
- サクラチップのドロップ数が1増える
- サクラチップ300枚で交換
- クエスト: アンビリカル・ヘア を周回する
